# CleanerX — Optimizer

Open source optimization program for Windows.

**Features:**
**•** Stops unnecessary services when started.\n
**•** Clears temp folders when started.
**•** Clears RAM when RAM usage exceeds 60%

**Requirements:**
.NET Framework 2.0
