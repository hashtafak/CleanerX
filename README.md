# CleanerX — Optimizer
Open source optimization program for Windows.

**Features:**<br />
**•** *Stops unnecessary services when started.*<br />
**•** *Clears temp folders when started.*<br />
**•** *Clears RAM when RAM usage exceeds 60%*<br />

**Requirements:**<br />
.NET Framework 2.0
